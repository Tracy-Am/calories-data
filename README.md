Calories Data Analysis (SQL Project)
Project Overview

This project analyzes a calories dataset using SQL to explore patterns in physical activity and calories burned. The goal of the analysis is to demonstrate data querying, filtering, aggregation, and analytical thinking using SQL.

The dataset contains information about users, their exercise duration, heart rate, body temperature, and the number of calories burned.

This project highlights how SQL can be used to extract insights from health and fitness data.

Objectives

The main objectives of this project were to:

Explore the dataset using SQL queries

Perform data filtering and sorting

Calculate summary statistics

Identify patterns between exercise variables and calories burned

Practice SQL skills used in real-world data analysis


SQL Skills Demonstrated

This project demonstrates the following SQL techniques:

Data filtering using WHERE

Aggregations using COUNT, AVG, SUM

Sorting results using ORDER BY

Grouping data using GROUP BY

Using logical conditions for analysis

Extracting meaningful insights from structured data

Example Analysis Questions

Some of the questions explored in this project include:

What is the average number of calories burned?

How does exercise duration affect calories burned?

Which age groups burn the most calories?

Does heart rate correlate with calories burned?

What are the highest calorie-burning sessions?
